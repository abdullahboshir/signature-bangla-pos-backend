# Signature Bangla POS - Backend

> [!CAUTION]
> **Project Status: Legacy & Inactive**
> This repository is no longer under active development. It represents an earlier monolithic version of the system that has since evolved into a modern Domain-Driven Design (DDD) Monorepo architecture.

## Successor Project

This project has been succeeded and reimplemented as part of the **Manoxen Enterprise Platform**.

- **Successor:** [Manoxen](https://github.com/abdullahboshir/business-platform) (Private Monorepo)
- **Status:** Archived as historical and authorship proof.

---

## Technical Overview

This legacy backend provided the initial core logic for Point of Sale operations, including inventory management, sales tracking, and user authentication. It served as the architectural foundation for the evolution into the Manoxen system.
